# Management of Generic Chinese Headphones

## Light Indicator
- **Red** → Charging  
- **Blue** → Fully charged  

## Button Functions
- **Minus Button (−)**  
  - Long press → Decrease volume  
  - Short press → Previous song  

- **Circle Button (⭘)**  
  - Click → Pause/Play music  
  - Click → Answer/Hang up calls  
  - Long press → Power on/Power off  

- **Plus Button (+)**  
  - Long press → Increase volume  
  - Short press → Next song

- **8 hours** → Continuous music playback  
- **2–3 hours** → Short charge for prolonged use
